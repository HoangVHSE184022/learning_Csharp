0. SDK LÀ GÌ? .NET SDK LÀ GÌ?
- SDK - SOFTWARE DEVELOPMENT KIT - LÀ TỪ DÙNG CHUNG CHO MỌI NGÔN NGỮ LẬP TRÌNH
- NÓ LÀ BỘ CÔNG CỤ, BỘ ĐỒ CHƠI, CÁC CÔNG CỤ, CÁC TOOL, CÁC APP (ĐƯỢC HÃNG LÀM RA NGÔN NGỮ LẬP TRÌNH) DÙNG CHO DÂN DEV, GIÚP DÂN DEV LÀM APP
- LÀM APP THÌ CẦN 1 NGÔN NGỮ LẬP TRÌNH ĐỂ VIẾT CÁC LỆNH, ĐỂ RA LỆNH CHO MÁY LÀM ĐIỀU GÌ ĐÓ. DÙNG NN LẬP TRÌNH + ĐỒ CHƠI (SDK) SẼ GIÚP TẠO RA APP VÀ RUN APP
- MUỐN HỌC 1 NN LẬP TRÌNH NÀO ĐÓ ĐỂ VIẾT APP GÌ ĐÓ TRƯỚC TIÊN TA CẦN CÀI SDK.

* VỚI JAVA, TA CẦN JAVA SDK - GỌI TẮT LÀ JDK (JAVA DEVELOPMENT KIT)
* VỚI C# (# Sharp), VB.NET, VC++, .NET, F#: 4 NN LẬP TRÌNH RA ĐỜI KHOẢNG NĂM 2000-2001 BỞI MICROSOFT. MS XÀI CHUNG 1 SDK CHO CẢ 4, CHO NÊN SDK NÀY GỌI CHUNG LÀ .NET SDK

* LỊCH SỬ PHÁT TRIỂN CỦA SDK ĐI QUA NHIỀU TÊN GỌI, VERSION KHÁC NHAU
- KHỞI ĐẦU, .NET SDK TÊN LÀ: .NET FRAMEWORK (CHỈ CHẠY TRÊN WINDOWS)
			     .4.8X VERSION CUỐI

- SAU NÀY MS HỖ TRỢ ĐA NỀN (MACOS, LINUX) .NET SDK MANG TÊN .NET CORE

* SDK VS. RUNTIME
- SDK > RUNTIME
- CÀI SDK TỨC LÀ CÀI LUÔN RUNTIME. RUNTIME CÓ THỂ CÀI RIÊNG MÀ KO CẦN SDK (PHẦN DÙNG CHO DÂN DEV)
- RUNTIME: BỘ THƯ VIỆN, CÂU LỆNH, MÔI TRƯỜNG GIÚP APP MÌNH VIẾT RA TƯƠNG TÁC ĐƯỢC VỚI OS, CHẠY ĐƯỢC VỚI OS; NHỎ GỌN, KHÔNG CẦN PHẦN TOOL DÀNH CHO DÂN DEV

- SDK: KÈM THÊM RUNTIME, KÈM THÊM CÁC CÔNG CỤ CHO DÂN DEV: BIÊN DỊCH, DEBUG, GENERATE CODE DOCUMENT, KIỂM THỬ APP/CODE, ĐÒNG GÓI (.JAR .WAR) .DLL .EXE

* .NET CHIA RUNTIME THÀNH 3 LOẠI ỨNG VỚI 3 LOẠI APP (JAVA GỘP CHUNG): CONSOLE APP - APP CHẠY TRÊN DÒNG LỆNH
								      DESKTOP     - APP CHẠY TRÊN NỀN DESKTOP
								      WEB APP     - APP CHẠY QUA TRÌNH DUYỆT
